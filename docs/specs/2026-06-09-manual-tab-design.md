# 매뉴얼 탭(사용 가이드) 설계

> 작성일: 2026-06-09
> 상태: 승인됨 (구현 계획 대기)

## 목표

대시보드에 "매뉴얼" 탭을 추가해, 이 도구의 전체 기능 사용법을 한국어 가이드로 제공한다.
정적 임베드 HTML로 작성하며 백엔드/엔드포인트는 추가하지 않는다.

## 결정 사항 (확정)

- **콘텐츠 방식**: 정적 임베드 HTML — index.html의 새 탭 패널에 직접 작성. 백엔드 불필요.
- **범위**: 전체 기능 가이드(모든 탭·기능 + 멀티서버 운영 + 안전 주의사항).

## 배경 (현재 구조)

- 탭은 `nav.tabs > button.tab[data-tab=...]`, 패널은 `section.tabpanel[data-panel=...]`.
- `showTab(name)`/`initTabs()`(app.js)는 임의 패널 이름을 처리한다: 패널이 존재하면 전환,
  없으면 overview로 폴백. 즉 새 탭/패널만 추가하면 **JS 변경 없이** 전환이 동작한다.
- 카드(`.card`)·`.dim`·`code` 등 기존 스타일을 재사용한다.

## 컴포넌트 설계

### 1) 탭 추가
`nav.tabs`(개요/설정/목적지/복원) 끝에 추가:
```html
<button class="tab" data-tab="manual">매뉴얼</button>
```

### 2) 패널 추가
`section.tabpanel[data-panel="manual"] hidden` 안에 카드 7개(아래 콘텐츠). 마지막 패널로 배치.

### 3) 콘텐츠 (카드 단위, 한국어)
1. 시작하기(최초 설정): `.env`(REMOTE_NAME·HOST_TAG), `secrets/repo-pass`(⚠️ 분실 시 복구
   불가), `rclone.conf`, `ALLOW_REPO_INIT`.
2. 개요 탭: 지금 백업 / 스냅샷 / 실행 이력.
3. 설정 탭: 보존일수·업로드제한·스케줄·무결성검증·DB일관성 / 제외 규칙 / 백업 대상 경로·DB
   작업(추가·삭제) / 알림 webhook.
4. 목적지 탭: 목적지 추가(간편) / rclone 공식 설정·GUI·CLI·터미널 / 백업 저장 위치 —
   원격 경로 변경, 원격 전환(이동 vs 채택).
5. 복원 탭: 스냅샷·경로 모달 선택·비밀번호 재확인·결과 다운로드(라이브 데이터 미덮어씀).
6. 멀티 서버 운영: HOST_TAG로 서버별 자동 분리, 같은 경로 공유 시 주의(공유 저장소·잠금).
7. 안전·주의사항: repo-pass 분실=복구불가, 마이그레이션 안전 불변식(복사·검증 전 전환·삭제
   없음), 동시 실행 차단(게이트).

### 4) 스타일
`engine/web/ui/style.css`에 매뉴얼 가독성용 소폭 규칙 추가: 패널 내 `h3`(섹션 소제목) 간격,
`ul/ol` 들여쓰기, `.manual-note`(경고 박스) 정도. 기존 `.card`/`.dim` 재사용.

## 데이터 흐름

없음(정적 콘텐츠). 탭 클릭 → 기존 `showTab` → 패널 표시.

## 오류 처리

해당 없음(정적). 캐시버스트 스탬프만 갱신해 새 마크업이 즉시 반영되게 한다.

## 테스트

- 수동: 매뉴얼 탭 클릭 → 콘텐츠 표시, 다른 탭 전환 정상, 새로고침 후 해시(#manual) 유지.
- JS 문법 검사(app.js 변경 시; 없으면 생략 가능).

## 변경 파일

- `engine/web/ui/index.html` — 탭 버튼 + 매뉴얼 패널(콘텐츠) + 캐시버스트 `?v=`
- `engine/web/ui/style.css` — 매뉴얼 가독성 규칙
- `engine/web/ui/app.js` — BUILD 스탬프만 갱신(로직 변경 없음)

## 범위 밖 (YAGNI)

- 마크다운 렌더링·별도 엔드포인트.
- 다국어.
- 검색·목차 자동생성(섹션이 적어 불필요; 필요 시 후속).
